늘어나고 있다. 이를 고려해 보았을 때, 참여자 네트워크를 통해서 행성 자원
소유 기대권을 공증하고 이의 영속적인 증거가 보장되는 플랫폼이
만들어진다면 많은 거래 희망자가 생길 거라고 예측된다.**
